# SHE-SHIELD

SHE-SHIELD is a proactive women’s safety application designed to provide automated emergency assistance without requiring manual user intervention.

## Key Features
- Zero-action safety mechanism for critical situations
- Real-time GPS location tracking
- Route deviation detection for identifying abnormal movement
- Automatic SOS alert system to notify emergency contacts

## Tech Stack
- Kotlin (Android)
- Firebase Authentication
- Firestore Database

## System Overview
The application continuously monitors user location and applies deviation detection logic to identify potential risk scenarios. Upon detecting anomalies, the system triggers automated alerts to pre-configured emergency contacts.

## Future Enhancements
- Integration of ML-based anomaly detection for improved accuracy
- Real-time analytics for user safety patterns
